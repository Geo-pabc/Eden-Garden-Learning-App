# Installation and Setup Guide

## Prerequisites
Before you begin, ensure you have met the following requirements:
- Node.js and npm installed (minimum version: Node.js 14.x)
- A code editor like VSCode or any other of your preference

## Installation Steps
1. **Clone the repository:**  
   Run the following command to clone the repository:
   ```
   git clone https://github.com/Geo-pabc/Eden-Garden-Learning-App.git
   ```

2. **Navigate to the project directory:**  
   ```
   cd Eden-Garden-Learning-App
   ```

3. **Install dependencies:**  
   Run the following command to install all necessary dependencies:
   ```
   npm install
   ```

4. **Start the application:**  
   You can start the application using:
   ```
   npm start
   ```
   This will launch the app in your default web browser at `http://localhost:3000`.

5. **Open the code in your editor:**  
   Feel free to open the project in your preferred code editor and start coding!

## Troubleshooting
- If you encounter any issues, make sure to check the console for errors and ensure all dependencies are installed properly.
